Para poder ejecutar programas Python con graficas usando Dash hace falta:

*Tener instalado Python 3
*Tener instalado pip
*Poner Python en el path de Windows, para que lo reconozca cuando se ejecute
*Instalar plotly, con:
	pip install plotly [enter]
*Instalar ipywidgets, con:
	pip install ipywidgets [enter]
*Instalar Dash, con:
	pip install dash [enter]
------------------------------------------
*Para ejecutar el programa de representacion por potencia,

En Windows:
	*Inicio -> escribir cmd [enter]
	*En la ventana de comando, usar el comando cd para situarse en el directorio donde están juntos todos los archivos y programas
	*En la ventana de comando escribir,
		python represent_Power.py [longitud de ventana] "[archivo]" [enter]
		Por ejemplo:
			python represent_Power.py 60 "SMM 2024-06-7 al 8.csv" [enter]
	*La instrucción completa para visualizar los resultados del archivo [archivo] en un navegador escribiendo http://127.0.0.1:[puerto] sería:
		python represent_Power.py [longitud de ventana] "[archivo]" [puerto] [enter]
		Por ejemplo:
			python represent_Power.py 60 "SMM 2024-06-7 al 8.csv" 8050 [enter]
	
En Linux:
	*En la carpeta donde esté la aplicación abrir un terminal
	*En la ventana del terminal activar el entorno Conda apropiado
		Por ejemplo:
			conda activate tf
	*En la ventana del terminal escribir,
		python represent_Power.py [longitud de ventana] "[archivo]" [enter]
		Por ejemplo:
			python represent_Power.py 60 "SMM 2024-06-7 al 8.csv" [enter]
	*La instrucción completa para visualizar los resultados del archivo [archivo] en un navegador escribiendo http://127.0.0.1:[puerto] sería:
		python represent_Power.py [longitud de ventana] "[archivo]" [puerto] [enter]
		Por ejemplo:
			python represent_Power.py 60 "SMM 2024-06-7 al 8.csv" 8050 [enter]
	
*Cuando termina, se abre navegador de internet en la siguiente dirección:
	http://127.0.0.1:[puerto] [enter]
	Por ejemplo, si el puerto es 8050,
		http://127.0.0.1:8050
